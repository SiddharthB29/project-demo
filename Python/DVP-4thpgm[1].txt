4.a)write a python program to demonstrate how to draw bar plot using matplotlib

CODE:

import matplotlib.pyplot as plt

# Sample data
categories = ['Category A', 'Category B', 'Category C', 'Category D']
values = [23, 45, 56, 78]

# Create the bar plot
plt.bar(categories, values, color='skyblue')

# Add labels and title
plt.xlabel('Categories')
plt.ylabel('Values')
plt.title('Bar Plot Example')

# Show the plot
plt.show()

Explanation:

pip install matplotlib(before execution)
plt.bar() is used to create a bar plot, 
with the categories as the x-axis labels and the values as the heights of the bars.
plt.xlabel() and plt.ylabel() are used to label the x and y axes.
plt.title() adds a title to the plot.
plt.show() is used to display the plot.


4.b)Write a Python program to Demonstrate how to Draw a Scatter Plot using Matplotlib. 

CODE:

import matplotlib.pyplot as plt

# Sample data
x = [5, 7, 8, 7, 2, 17, 2, 9, 4, 11]
y = [99, 86, 87, 88, 100, 86, 103, 87, 94, 78]

# Create the scatter plot
plt.scatter(x, y, color='red', marker='o')

# Add labels and title
plt.xlabel('X-axis values')
plt.ylabel('Y-axis values')
plt.title('Scatter Plot Example')

# Show the plot
plt.show()



Explanation:

plt.scatter(x, y) creates a scatter plot with x values on the x-axis and y values on the y-axis.
color specifies the color of the points, and marker defines the style of the marker (in this case, circles 'o').
plt.xlabel(), plt.ylabel(), and plt.title() are used to add labels and a title to the plot.
plt.show() displays the scatter plot.
